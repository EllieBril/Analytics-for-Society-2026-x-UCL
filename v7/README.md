# V7 Evidence Layer

This folder contains the V7 cluster-level modelling work. It is retained because it adds useful secondary evidence, but it is not the canonical project backend.

## What V7 is good for

- Showing that a stronger validation strategy was used than in V6.
- Highlighting school-context features that may be associated with stronger or weaker SES slopes.
- Supporting the `Evidence & Limits` part of the dashboard and written narrative.

## What V7 is not for

- It is not the official school recommender.
- It is not the official segmentation engine.
- It is not the country-trend engine.
- It should not be used to make deterministic claims for one named school.

## Why it remains secondary

- It is 2022-only.
- It is cluster-level rather than the official school-profile layer.
- Several feature names are still questionnaire-coded and require codebook translation.

Use the local 12-step pipeline and `outputs/` for official competition artifacts.
